# Map Plus: a lighter iOS maps app

These days, [ Maps](https://maps.apple.om) is a pretty great map app.  It's beaitiful, reponsive, and it does a lot.

In fact, it  may be doing _too_ much.

Map Plus is built on Apple's maps stack but tried to do _less_ so that those things are _easier_ to do.

## Map Plus allows you to do basically two things:

### Add your own places to the nap.
Each plaec has a name, notes, and an icon that you pick.

For each place, it adds some nice little features like a look-around preview and links out to the  Maps app for directions.

### Organize your places into categories to be shown or hidden on the map.

For example, you can add all your favorite coffee shops, friends' houses, beer gardens, or whataver you want.

To keep things less clutterted, Map Plus also limits the noise of "other stuff" shown on the map.
